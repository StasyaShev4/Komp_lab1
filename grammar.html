Грамматика

Определим грамматику структур языка PHP G[‹Struct›] в нотации Хомского с продукциями P:

1) <Struct> → 'struct' <NameStruct>
2) <NameStruct> → <Letter> <NameStruct> | '{' <Fields>
3) <Fields> → 'string' <Field>
            | 'int' <Field>
            | 'bool' <Field>
            | 'float' <Field>
            | 'array' <Field>
            | '}' <Final>
4) <Field> → '$' <NameVariable>
5) <NameVariable> → <Letter> <NameVariable> | ';' <Fields>
6) <Final> → ';'

Следуя введенному формальному определению грамматики, представим G[<Struct>] ее составляющими:

  • Z = <Struct>;
  • VT = { struct, string, int, bool, float, array, {, }, ;, $, a, b, ..., z, A, B, ..., Z };
  • VN = { <Struct>, <NameStruct>, <Fields>, <Field>, <NameVariable>, <Letter>, <Final> };
