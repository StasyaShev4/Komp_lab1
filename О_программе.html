О программе

Программный продукт представляет собой текстовый редактор с расширяемым функционалом, предназначенный для анализа исходного кода и дальнейшего развития до полноценного языкового процессора.

Быстрая навигация

  • GUI интерфейс
  • Лексический анализатор
  • Синтаксический анализатор
  • Технологии

GUI интерфейс

Разработан графический интерфейс в виде текстового редактора.

  • Основное меню
  • Панель инструментов
  • Область редактирования текста
  • Область вывода результатов

Функциональность:

  • Создание, открытие и сохранение файлов
  • Редактирование текста (копирование, вставка, отмена и др.)
  • Справка о программе
  • Подтверждение выхода при несохранённых данных

Лексический анализатор

Лексический анализатор предназначен для разбиения входного текста на лексемы.

Поддерживаемые элементы:

  • Ключевые слова: struct, string, int, bool, array, float
  • Идентификаторы
  • Переменные PHP (начинаются с $)
  • Разделители: { } ;

Пример:

struct Employee {
    string $name;
    int $salary;
};

Синтаксический анализатор

Синтаксический анализатор выполняет проверку структуры программы на основе заданной грамматики.

Особенности:

  • Автоматная грамматика
  • Метод анализа на основе конечного автомата
  • Обработка ошибок методом Айронса

Пример структуры:

struct User {
    string $username;
    int $age;
};

Используемые технологии

  • C#
  • WinForms
  • Visual Studio
