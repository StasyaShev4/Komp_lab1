Разработка пользовательского интерфейса (GUI)

Проект представляет собой разработку оконного приложения — текстового редактора, предназначенного для последующего расширения функционала до полноценного языкового процессора для анализа исходного кода.

Программа доработана в рамках курсовой работы.

Рисунок 6 – панель инструментов

Область редактирования

Область редактирования — это текстовый редактор для ввода, изменения и сохранения файлов. Здесь можно вводить текст, применять форматирование и выполнять другие операции.

Меню «Файл» управляет файлами: создание, открытие, сохранение. Меню «Правка» содержит команды копирования, вставки, вырезания, отмены и другие инструменты для работы с текстом. Кнопка пуск (зеленый треугольник) (см. рис. 6) запускает синтаксический анализатор. Результат запускак отобразится в Области отображения результатов.

Область отображения результатов

Область вывода результатов представляет собой таблицу со следующими столбцами:

  • Фрагмент — сдесь отображаются лексемы
  • Местоположение — позиция лексемы в тексте
  • Описание — описание ошибки в тексте

Реализована навигация по ошибкам: при клике курсор автоматически переходит к месту ошибки (см. рис. 7).
