## Контекстно-зависимые условия:

По варианту задания подходит: 
Правило 1 **(уникальность идентификаторов)**: Проверить, что имя не было объявлено ранее в той же области видимости.

Так как только это правило можно реализовать для строки выше. Были приведены доработки: 

```
struct U {
    int $iв = 314;
    float $ab = 3.14;
    string $name = "Aboba";
    bool $isAdmin = false;
};
```

Для таких строк можно реализовать еще правила: 

Правило 2 **(совместимость типов)**: Проверить, что тип инициализирующего значения соответствует объявленному типу (для констант/переменных).

Правило 3 **(допустимые значения)**: Проверить, что значение находится в допустимых пределах (для числовых типов).

## Структура AST:

### 1. Описание типов узлов

Каждый узел дерева представляет отдельную синтаксическую конструкцию и содержит информацию о своих атрибутах и дочерних элементах.

| Тип узла       | Описание                  | Атрибуты                         | Дочерние узлы   |
| -------------- | ------------------------- | -------------------------------- | --------------- |
| StructDeclNode | Объявление структуры      | name                             | FieldDeclNode[] |
| FieldDeclNode  | Объявление поля структуры | name, type, value (необязателен) | —               |

## Дополнительное задание
Для визуализации абстрактного синтаксического дерева (AST) в работе были использованы два подхода: текстовое представление и графическое отображение.
### Использованные графические средства

Для построения графического представления AST была использована система визуализации графов Graphviz. Данный инструмент позволяет описывать структуру графа в текстовом формате (язык DOT), после чего автоматически генерировать наглядное изображение в виде ориентированного графа.

Сгенерированный DOT-файл обрабатывается утилитой `dot`, входящей в состав Graphviz, с последующим формированием изображения в формате PNG. Полученное изображение автоматически открывается для просмотра.

#### Пример: 

```
struct User {
    int $id = 1;
    string $name = "Admin";
};
struct Product {
    float $price = 99.99;
    bool $inStock = true;
    array $tags = [];
};
```
