# Разработка пользовательского интерфейса (GUI) для языкового процессора

### **Цель работы.**

Создание кроссплатформенного графического интерфейса (GUI) для языкового процессора в виде специализированного текстового редактора.

## Описание проекта:
Проект представляет собой разработку оконного приложения — текстового редактора, предназначенного для последующего расширения функционала до полноценного языкового процессора для анализа исходного кода.

В ходе лабораторной работы был разработан GUI интерфейс состоящий из таких элементов как: 
1. **Основное меню** которое включает команды управления файлами, правкой текста и помощь пользователю
2. **Панель инструментов** что обеспечивает быстрый доступ к наиболее востребованным командам ("создать", "открыть", "сохранить", стандартные операции с буфером обмена и др.)
3. **Область ввода/редактирования текста** — основной рабочий компонент приложения, предназначенный для внесения и изменения содержимого файла
4. **Область вывода результатов обработки** — отображение итогов работы встроенных алгоритмов анализа (предназначено исключительно для чтения)

Все элементы интерфейса динамически масштабируемые. 

Также были разработаны обязательный функции дня текстового редактора.  

1. Работа с файлами:
	- Создание нового файла.    
	- Открытие существующего файла.        
	- Сохранение текущего файла.        
	- Сохранение файла под новым именем.
	
2. Функции редактирования текста, включающие стандартные операции:
    - Отмена последнего действия      
    - Повтор отменённого действия
    - Копирование выделенного текста
    - Вырезание выделенного фрагмента
    - Вставка скопированного текста      
    - Выделение всего текста
    
3. Справка, содержащая сведения о программе.
    
4. Окно подтверждения выхода из программы, которое появляется при закрытии приложения, если данные в текущем файле не были сохранены.
