## Решение 3 задач:
### 1) Гласные (без а)
##### Описание:
 Построить РВ для поиска всех российских гласных букв кроме буквы а или А. 
##### Регулярное выражение с пояснением каждого обозначения.
[еёиоуюыэяЕЁИОУЮЫЭЯ]
- `[` `]` — **символьный класс** (любой символ из списка)
- `еёиоуюыэя` — строчные русские гласные (без `а`)
- `ЕЁИОУЮЫЭЯ` — заглавные русские гласные (без `А`)
##### Примеры строк, которые должны находиться:
Найдёт: е, ё, и, о, у, ю, ы, э, я
##### Примеры строк, которые не должны находиться:
Не найдёт: а, А, б, д, М (и другие согласные).

### 2) Акронимы (NASA, N.B.A.)
##### Описание:
 Построить РВ для поиска акронимов (заглавные буквы, возможно с точками, например N.B.A. или NASA).
##### Регулярное выражение с пояснением каждого обозначения.
\b(?:[A-ZА-ЯЁ]\.){2,}|[A-ZА-ЯЁ]{2,}\b
- `\b` — граница слова
- `(?: ... )` — **группа без сохранения**
- `[A-ZА-ЯЁ]` — заглавная буква (латиница + кириллица)
- `\.` — точка
- `{2,}` — **минимум 2 повторения**
##### Примеры строк, которые должны находиться:
NASA, USA, N.A.S.A., U.S.A., ФСБ, ВОЗ, Ф.С.Б. ,COVID
##### Примеры строк, которые не должны находиться:
NaSa, nasa, A, N., A.B, Hello, Test123

### 3) URL
##### Описание:
 Построить РВ, описывающее URL-ссылку на веб-страницу на латинице с поддержкой поддоменов и портов (с различными протоколами HTTP, HTTPS, FTP).
##### Регулярное выражение с пояснением каждого обозначения.
\b(http|https|ftp)(:\/\/)(([a-zA-Z0-9-]+\.)+[a-zA-Z]{2,})(:\d+)?(\/[^\s]*)?
- `\b` — граница слова
- `(http|https|ftp)` — протокол
- `:\/\/` — разделитель
- `([a-zA-Z0-9-]+\.)+` — поддомены
- `[a-zA-Z]{2,}` — домен верхнего уровня
- `(:\d+)?` — порт (необязательно)
- `(\/[^\s]*)?` — путь (необязательно)
##### Примеры строк, которые должны находиться:
http://example.com  
https://google.com  
ftp://files.server.net    
https://api.site.ru:8080  
http://test.org:80    
https://sub.domain.com/path  
http://site.ru/index.html
##### Примеры строк, которые не должны находиться:
htp://site.com
http://site
http://.com 
http://site.c   
http://site,com
